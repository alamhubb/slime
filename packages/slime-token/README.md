# slime-token

Token type definitions and lexical rules for Slime parser.

## Installation

```bash
npm install slime-token
```

## Usage

```typescript
import { SlimeJavascriptTokenType, SlimeTypescriptContextualKeywordTokenTypes } from 'slime-token'
```
