# slime-generator

AST to code generator for Slime parser. Converts ESTree-compatible AST back to source code.

## Installation

```bash
npm install slime-generator
```

## Usage

```typescript
import SlimeGenerator from 'slime-generator'

const code = SlimeGenerator.generate(ast)
```
