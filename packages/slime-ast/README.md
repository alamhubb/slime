# slime-ast

ESTree-compatible AST type definitions and factory methods for Slime parser.

## Installation

```bash
npm install slime-ast
```

## Usage

```typescript
import { SlimeAstUtil, SlimeAstTypeName } from 'slime-ast'

// Create AST nodes
const identifier = SlimeAstUtil.createIdentifier('foo')
```
